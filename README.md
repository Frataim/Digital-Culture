# Digital-Culture

## Введение

Digital-Culture - это приложение создано для выполнения задач в области цифровизации искусства.
Музеи, галереи и театры сегодня стремяться стать частью цифровой действительности, войти в новый век высоких технологий. Но им нужна твоя помощь, программист!
Любое государственное учреждение в области искусства может оставить задачу и любой волонтер-программист может ее выбрать и выполнить!

### На проекте были выполнены следующие задачи:
- регистрация и авторизация пользователя (заказчик/исполнитель)
- личный кабинет (заказчик/исполнитель)
- чат между заказчиком и исполнителем
- добавление и удаление задач
- статус выполнения задачи
- фильтр задач
- возможность поставить рейтинг исполнителю
- визуализация приложения
